This is an ARITH (simple Arithmetics) task from  SPHERE Online Judge (SPOJ).
link: https://www.spoj.com/problems/ARITH/

All the input data (https://contest.felk.cvut.cz/00cerc/solved/arith.in) is shown in input.txt
When running this task (arith.php) the output will be saved in output.txt.
The correct answers are saved in check.txt

To see if my answers differ from correct answers (https://contest.felk.cvut.cz/00cerc/solved/arith.out),
you can use unix command to check file difference (diff output.txt check.txt). It will show which lines don't match.
